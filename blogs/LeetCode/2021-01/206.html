Jzt 2021-01-03 链表

反转一个单链表

# 题目

206. 反转链表

难度简单

反转一个单链表。

示例:

输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL

进阶: 你可以迭代或递归地反转链表。你能否用两种方法解决这道题？

# 思路

参考题解

思路一：借用列表存储数据，空间复杂度O(N)

思路二：迭代。两个指针，pre指向上一个元素，cur指向当前元素，然后用cur->next = pre,就可以实现链表逆转，注意需要用临时变量tmp = cur->next;然后cur = tmp，这样cur才能往下移动。空间复杂度O(1)

思路三：递归，不太好理解。

cur.next.next = cur，也就是用下一个节点next指向当前节点，再逐级返回末端的“头结点”。

cur.next = null;这样新链表的最后一个节点的next指针指向空。

# 代码一

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        ArrayList<Integer> list = new ArrayList<>();
        ListNode p = head;
        while(p!=null){
            list.add(p.val);
            p = p.next;
        }
        p = null;
        for(int i = list.size() - 1;i >=0; i--){
            ListNode s = new ListNode(list.get(i));
            s.next =null;
            if(p == null){
                head = s;
                p = head;
            }else{
                p.next = s;
                p = s;
            }
        }
        return head;
    }
}

# 代码二

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode pre = null;
        ListNode cur = head;
        ListNode tmp;
        while(cur!=null){
            tmp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = tmp;
        }
        return pre;
    }
}

# 代码三

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        if(head == null || head.next == null){
            return head;
        }
        ListNode cur = reverseList(head.next);
        head.next.next = head;
        head.next = null;
        return cur;
    }
}
